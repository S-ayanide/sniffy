# sniffy

> A GitHub App built with [Probot](https://github.com/probot/probot) that An app to sniff the size of PRs as well as attach labels depending on area the PR is submitted

## Setup

```sh
# Install dependencies
npm install

# Run with hot reload
npm run build:watch

# Compile and run
npm run build
npm run start
```
